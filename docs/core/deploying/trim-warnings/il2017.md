# IL2017: Could not find property on type in trimmer file

## Cause

A property specified for in a trimmer data format file could not be found.

## Rule description

A property specified in a [trimmer data format](https://github.com/mono/linker/blob/main/docs/data-formats.md)
file could not be found in the type matching the `fullname` argument that was passed to
the parent of the `property` element.

## Example

```XML
<!-- IL2017: Could not find property 'NonExistentProperty' on type 'MyType' -->
<linker>
  <assembly fullname="MyAssembly">
    <type fullname="MyType">
      <property name="NonExistentProperty" />
    </type>
  </assembly>
</linker>
```
