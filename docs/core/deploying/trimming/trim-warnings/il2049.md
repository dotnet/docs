# Trim Warning IL2049: : Unrecognized internal attribute '{0}'

The internal attribute name '{0}' being used in the xml is not supported by the linker, check the spelling and the supported internal attributes.

## Cause

An internal attribute name specified in a custom attribute annotations file is not
supported by the trimmer.

## Example

```XML
<!-- IL2049: Unrecognized internal attribute 'InvalidInternalAttributeName' -->
<linker>
  <assembly fullname="MyAssembly">
    <type fullname="MyType">
      <method name="MyMethod">
        <attribute internal="InvalidInternalAttributeName" />
      </method>
    </type>
  </assembly>
</linker>
```
