# ICorDebugEval::Abort Method

Aborts the computation this ICorDebugEval object is currently performing.

## Syntax

```cpp
HRESULT Abort ();
```

## Remarks

If the evaluation is nested and it is not the most recent one, the `Abort` method may fail.
