# IMetaDataEmit::SetRVA Method

Sets the relative virtual address of the specified method.  
  
## Syntax  
  
```cpp  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,
    [in]  ULONG        ulRVA
);  
```  
  
## Parameters  

 `md`  
 [in] The token for the target method or method implementation.  
  
 `ulRVA`  
 [in] The address of the code or data area.
