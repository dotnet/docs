# IMetaDataTables::GetBlobHeapSize Method

Gets the size, in bytes, of the binary large object (BLOB) heap.

## Syntax

```cpp
HRESULT GetBlobHeapSize (
    [out] ULONG     *pcbBlobs
);
```

## Parameters

 `pcbBlobs`
 [out] A pointer to the size, in bytes, of the BLOB heap.
