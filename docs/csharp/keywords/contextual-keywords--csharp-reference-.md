# Contextual Keywords (C# Reference)
A contextual keyword is used to provide a specific meaning in the code, but it is not a reserved word in C#. The following contextual keywords are introduced in this section:  
  
|Keyword|Description|  
|-------------|-----------------|  
|[add](../keywords/add--csharp-reference-.md)|Defines a custom event accessor that is invoked when client code subscribes to the event.|  
|[async](../keywords/async--csharp-reference-.md)|Indicates that the modified method, lambda expression, or anonymous method is asynchronous.|  
|[await](../keywords/await--csharp-reference-.md)|Suspends an async method until an awaited task is completed.|  
|[dynamic](../keywords/dynamic--csharp-reference-.md)|Defines a reference type that enables operations in which it occurs to bypass compile-time type checking.|  
|[get](../keywords/get--csharp-reference-.md)|Defines an accessor method for a property or an indexer.|  
|[global](../keywords/global--csharp-reference-.md)|Specifies the default global namespace, which is otherwise unnamed.|  
|[partial](../keywords/partial--type---csharp-reference-.md)|Defines partial classes, structs, and interfaces throughout the same compilation unit.|  
|[remove](../keywords/remove--csharp-reference-.md)|Defines a custom event accessor that is invoked when client code unsubscribes from the event.|  
|[set](../keywords/set--csharp-reference-.md)|Defines an accessor method for a property or an indexer.|  
|[value](../keywords/value--csharp-reference-.md)|Used to set accessors and to add or remove event handlers.|  
|[var](../keywords/var--csharp-reference-.md)|Enables the type of a variable declared at method scope to be determined by the compiler.|  
|[where](../keywords/where--generic-type-constraint---csharp-reference-.md)|Adds constraints to a generic declaration. (See also [where](../keywords/where-clause--csharp-reference-.md)).|  
|[yield](../keywords/yield--csharp-reference-.md)|Used in an iterator block to return a value to the enumerator object or to signal the end of iteration.|  
  
 All query keywords introduced in C# 3.0 are also contextual. For more information, see [Query Keywords (LINQ)](../keywords/query-keywords--csharp-reference-.md).
