# Jump Statements (C# Reference)
Branching is performed using jump statements, which cause an immediate transfer of the program control. The following keywords are used in jump statements:  
  
-   [break](../keywords/break--csharp-reference-.md)  
  
-   [continue](../keywords/continue--csharp-reference-.md)  
  
-   [goto](../keywords/goto--csharp-reference-.md)  
  
-   [return](../keywords/return--csharp-reference-.md)  
  
-   [throw](../keywords/throw--csharp-reference-.md)
