# Compiler Warning (level 4) CS1591
[!INCLUDE[csharpbanner](../../../includes/csharpbanner.md)]

Missing XML comment for publicly visible type or member 'Type_or_Member'  
  
 The [/doc](../../../csharp/language-reference/compiler-options/doc-csharp-compiler-options.md) compiler option was specified, but one or more constructs did not have comments.  
  
 The following sample generates CS1591:  
  
```  
// CS1591.cs  
// compile with: /W:4 /doc:x.xml  
  
/// text  
public class Test  
{  
   // /// text  
   public static void Main()   // CS1591, remove "//" from previous line  
   {  
   }  
}  
```
