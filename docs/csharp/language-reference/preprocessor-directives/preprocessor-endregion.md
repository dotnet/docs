# #endregion (C# Reference)
`#endregion` marks the end of a [#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) block. For example:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
