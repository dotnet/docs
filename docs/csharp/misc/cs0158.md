# Compiler Error CS0158
The label 'label' shadows another label by the same name in a contained scope  
  
 A label in an inner scope hides a label with the same name in an outer scope. For more information, see [goto](../../csharp/language-reference/keywords/goto.md).  
  
 The following sample generates CS0158:  
  
```csharp  
// CS0158.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         goto lab1;  
         lab1:  
         {  
            lab1:  
            goto lab1;   // CS0158  
         }  
      }  
   }  
}  
```
