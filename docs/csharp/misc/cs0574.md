# Compiler Error CS0574
Name of destructor must match name of class  
  
 The name of a destructor must be the class name preceded by a tilde (~).  
  
 The following sample generates CS0574:  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
