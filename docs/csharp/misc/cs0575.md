# Compiler Error CS0575

Only class types can contain destructors  
  
 A [struct](../language-reference/builtin-types/struct.md) cannot contain a finalizer.  
  
 The following sample generates CS0575:  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
