# Compiler Error CS0575
Only class types can contain destructors  
  
 A [struct](../../csharp/language-reference/keywords/struct.md) cannot contain a destructor.  
  
 The following sample generates CS0575:  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
