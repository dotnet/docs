# Compiler Error CS0584

Internal Compiler Error: stage 'stage' symbol 'symbol'
  
 Try to determine whether the compiler is failing because of its inability to parse unexpected syntax. If that is not the case, see [Visual Studio feedback options](/visualstudio/ide/feedback-options).

> [!NOTE]
> This compiler error is no longer used in Roslyn.
