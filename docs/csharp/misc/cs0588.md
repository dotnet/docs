# Compiler Error CS0588

Internal Compiler Error: stage 'LEX'

 Try to determine whether the compiler is failing because of its inability to parse unexpected syntax. If this is not the case, see [Visual Studio feedback options](/visualstudio/ide/feedback-options).

> [!NOTE]
> This compiler error is no longer used in Roslyn.
