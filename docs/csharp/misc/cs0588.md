# Compiler Error CS0588
Internal Compiler Error: stage 'LEX'  
  
 Try to determine whether the compiler is failing because of its inability to parse unexpected syntax. If this is not the case, try [Talk to Us](https://docs.microsoft.com/visualstudio/ide/talk-to-us).
