# Compiler Error CS0631
ref and out are not valid in this context  
  
 The declaration for an [indexer](../../csharp/programming-guide/indexers/index.md) cannot include the use of [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameters.  
  
## Example  
 The following sample generates CS0631:  
  
```csharp  
// CS0631.cs  
public class MyClass  
{  
    public int this[ref int i]   // CS0631  
    // try the following line instead  
    // public int this[int i]  
    {  
        get  
        {  
            return 0;  
        }  
    }  
}  
  
public class MyClass2  
{  
    public static void Main()  
    {  
    }  
}  
```
