# Compiler Error CS0663
Cannot define overloaded methods that differ only on ref and out.  
  
 Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.  
  
 The following sample generates CS0663:  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
