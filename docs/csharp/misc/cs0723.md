# Compiler Error CS0723
Cannot declare variable of static type 'type'  
  
 Instances of static types cannot be created.  
  
 The following sample generates CS0723:  
  
```  
// CS0723.cs  
public static class SC  
{  
}  
  
public class CMain  
{  
   public static void Main()  
   {  
      SC sc = null;  // CS0723  
   }  
}  
```
