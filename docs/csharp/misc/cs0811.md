# Compiler Error CS0811
The fully qualified name for 'name' is too long for debug information. Compile without '/debug' option.  
  
 There are size constraints on variable and type names in debug information.  
  
### To correct this error  
  
1.  If modifying the name is not possible, the only alternative is to compile without the [/debug](../../csharp/language-reference/compiler-options/debug-csharp-compiler-options.md) option.

    class Test  
    {  
        static int Main()  
        {  
            return 1;  
        }  
    }  
}
