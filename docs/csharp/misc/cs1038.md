# Compiler Error CS1038
\#endregion directive expected  
  
 A [#region](../../csharp/language-reference/preprocessor-directives/preprocessor-region.md) directive did not have a matching [#endregion](../../csharp/language-reference/preprocessor-directives/preprocessor-endregion.md) directive.  
  
 The following sample generates CS1038:  
  
```csharp  
// CS1038.cs  
#region testing  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
// CS1038  
// uncomment the next line to resolve  
// #endregion  
```
