# Compiler Error CS1508
Resource identifier 'identifier' has already been used in this assembly  
  
 In a compilation, the same identifier (***identifier***) was passed to two or more [/resource](../../csharp/language-reference/compiler-options/resource-csharp-compiler-options.md) or [/linkresource](../../csharp/language-reference/compiler-options/linkresource-csharp-compiler-options.md) compiler options.  
  
 For example, the following options would generate CS1508:  
  
```  
/resource:anyfile.bmp,DuplicatIdent /linkresource:a.bmp,DuplicatIdent  
```
