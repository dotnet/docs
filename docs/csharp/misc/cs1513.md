# Compiler Error CS1513

} expected  
  
 The compiler expected a closing curly brace (`}`) that was not found.  
  
 The following sample generates CS1513:  
  
```csharp  
// CS1513  
namespace y   // CS1513, no close curly brace  
{  
   class x  
   {  
      public static void Main()  
      {  
      }  
   }  
```
