# Compiler Error CS1514

{ expected  
  
 The compiler expected an opening curly brace (`{`) that was not found.  
  
 The following sample generates CS1514:  
  
```csharp  
// CS1514.cs  
namespace x  
// CS1514, no open curly brace  
```
