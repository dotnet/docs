# Compiler Error CS1541
Invalid reference option: 'symbol' â€” cannot reference directories  
  
 The compiler detected an attempt to specify a directory rather than a specific file. For example, when you use the [/reference](../../csharp/language-reference/compiler-options/reference-csharp-compiler-options.md) compiler option, you must specify a file; it is not possible to specify a directory.  
  
 For example, passing `/reference:c:\` to the compiler would generate CS1541.
