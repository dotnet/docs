# Compiler Error CS1563

Output 'output file' does not have any source files  
  
 The [**OutputAssembly**](../language-reference/compiler-options/output.md#outputassembly) compiler option was specified, but no source code files follow. You should either not use **OutputAssembly**, or you should specify the source code files following **OutputAssembly**.
