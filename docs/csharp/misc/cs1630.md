# Compiler Error CS1630
Invalid option 'option' for /errorreport; must be prompt, send, queue, or none  
  
 The command line option [/errorreport](../../csharp/language-reference/compiler-options/errorreport-compiler-option.md) must be followed by **prompt**, **send**, **queue**, or **none**, specifying what action you want taken when an internal compiler error occurs.
