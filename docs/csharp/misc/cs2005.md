# Compiler Error CS2005
Missing file specification for 'option' option  
  
 A [compiler option](../../csharp/language-reference/compiler-options/index.md) was only partially specified.  
  
 For example, when using [/recurse](../../csharp/language-reference/compiler-options/recurse-csharp-compiler-options.md), you must specify the file to search: **/recurse:***filename***.cs**.  
  
## Example  
 The following sample generates CS2005.  
  
```  
// CS2005.cs  
// compile with: /recurse:  
// CS2005 expected  
class x  
{  
   public static void Main() {}  
}  
```
