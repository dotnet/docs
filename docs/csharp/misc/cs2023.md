# Compiler Warning (level 1) CS2023

Ignoring /noconfig option because it was specified in a response file  
  
 The [-noconfig](../language-reference/compiler-options/miscellaneous.md#noconfig) compiler option was specified in a response file, which is not allowed.  
  
 This warning cannot be suppressed by the [**DisabledWarnings**](../language-reference/compiler-options/errors-warnings.md#disabledwarnings) option.
