# Compiler Error CS2024
Invalid file section alignment number '#'  
  
 An invalid value was passed to the [/filealign](../../csharp/language-reference/compiler-options/filealign-compiler-option.md) compiler option.  
  
 The following sample generates CS2024:  
  
```  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
