# #endregion (C# Reference)
`#endregion` marks the end of a [#region](../preprocessor-directives/sharpregion--csharp-reference-.md) block. For example:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
