# General POSDM Switches (POS for .NET v1.14 SDK Documentation)

The following general switches may be used between POSDM and a command on the command line.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><strong>Switch</strong></th>
<th><strong>Description</strong></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>/?</p></td>
<td><p>Displays <strong>posdm</strong> Help information.</p></td>
</tr>
<tr class="even">
<td><p>/output:file</p></td>
<td><p>Prints all output to file instead of to the console.</p></td>
</tr>
<tr class="odd">
<td><p>/machine:host</p></td>
<td><p>Uses host as the remote computer. Default is the local computer.</p></td>
</tr>
<tr class="even">
<td><p>/user:username</p></td>
<td><p>Uses user name as the user account. Prefix with domain separated by a backslash if the user domain is not the current one.</p></td>
</tr>
<tr class="odd">
<td><p>/password:password</p></td>
<td><p>Uses password with the specified user name account.</p></td>
</tr>
</tbody>
</table>
