# -- (Comment) (Entity SQL)

Entity SQL queries can contain comments. Two dashes (`--`) start a comment line.

## Syntax

```csharp
-- text_of_comment
```

## Arguments

 `text_of_comment`
 Is the character string that contains the text of the comment.
