# Method-Based Query Examples (LINQ to DataSet)

This section provides LINQ to DataSet programming examples in method-based query syntax that use the standard query operators. The <xref:System.Data.DataSet> used in these examples is populated by using the `FillDataSet` method, which is specified in [Loading Data Into a DataSet](loading-data-into-a-dataset.md). For more information, see [Standard Query Operators Overview (C#)](../../../csharp/linq/standard-query-operators/index.md) or [Standard Query Operators Overview (Visual Basic)](../../../visual-basic/programming-guide/concepts/linq/standard-query-operators-overview.md).

## In This Section

 [Projection](method-based-query-syntax-examples-projection.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.Select*> and <xref:System.Linq.Enumerable.SelectMany*> methods to query a <xref:System.Data.DataSet>.

 [Partitioning](method-based-query-syntax-examples-partitioning-linq.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.Skip*> and <xref:System.Linq.Enumerable.Take*> methods to query a <xref:System.Data.DataSet> and partition the results.

 [Ordering](method-based-query-syntax-examples-ordering-linq-to-dataset.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.OrderBy*>, <xref:System.Linq.Enumerable.OrderByDescending*>, <xref:System.Linq.Enumerable.Reverse*>, and <xref:System.Linq.Enumerable.ThenByDescending*> methods to query a <xref:System.Data.DataSet> and order the results.

 [Set Operators](method-based-query-syntax-examples-set-operators.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.Distinct*>, <xref:System.Linq.Enumerable.Except*>, <xref:System.Linq.Enumerable.Intersect*>, and <xref:System.Linq.Enumerable.Union*> operators to perform value-based comparison operations on sets of data rows.

 [Conversion Operators](method-based-query-syntax-examples-conversion-operators.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.ToArray*>, <xref:System.Linq.Enumerable.ToDictionary*>, and <xref:System.Linq.Enumerable.ToList*> methods to immediately execute a query expression.

 [Element Operators](method-based-query-syntax-examples-element-operators.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.First*> and <xref:System.Linq.Enumerable.ElementAt*> methods to get <xref:System.Data.DataRow> elements from a <xref:System.Data.DataSet>.

 [Aggregate Operators](method-based-query-syntax-examples-aggregate-operators.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.Average*>, <xref:System.Linq.Enumerable.Count*>, <xref:System.Linq.Enumerable.Max*>, <xref:System.Linq.Enumerable.Min*>, and <xref:System.Linq.Enumerable.Sum*> methods to query a <xref:System.Data.DataSet> and aggregate data.

 [Join](method-based-query-syntax-examples-join-linq-to-dataset.md)
 The examples in this topic demonstrate how to use the <xref:System.Linq.Enumerable.GroupJoin*> and <xref:System.Linq.Enumerable.Join*> methods to query a <xref:System.Data.DataSet>.
