# Install .NET Framework on Windows 11

.NET Framework 4.8 is included with Windows 11, and runs any .NET Framework 4.x app.

## .NET Framework 4.8.1

.NET 4.8.1 is supported on Windows 11, version 22H2 and newer operating systems.
