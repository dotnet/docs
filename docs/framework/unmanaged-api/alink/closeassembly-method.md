# CloseAssembly Method

Finalizes assembly operations. Call this method before beginning a new assembly or unbound module.  
  
## Syntax  
  
```cpp  
HRESULT CloseAssembly(  
    mdAssembly AssemblyID  
) PURE;  
```  
  
## Parameters  

 `AssemblyID`  
 ID of the assembly.  
  
## Return Value  

 Returns S_OK if the method succeeds.  
  
## Requirements  

 Requires alink.h.
