# ICorDebugCode::GetSize Method

Gets the size, in bytes, of the binary code represented by this "ICorDebugCode".

## Syntax

```cpp
HRESULT GetSize (
    [out] ULONG32    *pcBytes
);
```

## Parameters

`pcBytes`  
[out] A pointer to the size, in bytes, of the binary code that this `ICorDebugCode` object represents.
