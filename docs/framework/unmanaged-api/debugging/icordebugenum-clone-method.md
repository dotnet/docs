# ICorDebugEnum::Clone Method

Creates a copy of this ICorDebugEnum object.  
  
## Syntax  
  
```cpp  
HRESULT Clone (  
    [out] ICorDebugEnum **ppEnum  
);  
```  
  
## Parameters  

 `ppEnum`  
 [out] A pointer to the address of an `ICorDebugEnum` object that is a copy of this `ICorDebugEnum` object.
