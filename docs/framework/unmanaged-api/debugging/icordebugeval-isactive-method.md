# ICorDebugEval::IsActive Method

Gets a value that indicates whether this ICorDebugEval object is currently executing.  
  
## Syntax  
  
```cpp  
HRESULT IsActive (  
    [out] BOOL              *pbActive  
);  
```  
  
## Parameters  

 `pbActive`  
 [out] Pointer to a value that indicates whether this evaluation is active.
