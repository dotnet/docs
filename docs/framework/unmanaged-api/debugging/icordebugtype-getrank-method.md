# ICorDebugType::GetRank Method

Gets the number of dimensions in an array type.  
  
## Syntax  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## Parameters  

 `pnRank`  
 [out] A pointer to the number of dimensions.
