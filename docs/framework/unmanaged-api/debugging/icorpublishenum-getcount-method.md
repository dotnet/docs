# ICorPublishEnum::GetCount Method

Gets the number of items in the enumeration.  
  
## Syntax  
  
```cpp  
HRESULT GetCount (  
    [out] ULONG   *pcelt  
);  
```  
  
## Parameters  

 `pcelt`  
 [out] A pointer to the number of items in the enumeration.
