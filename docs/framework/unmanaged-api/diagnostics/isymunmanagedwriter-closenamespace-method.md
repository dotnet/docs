# ISymUnmanagedWriter::CloseNamespace Method

Closes the most recently opened namespace.  
  
## Syntax  
  
```cpp  
HRESULT CloseNamespace();  
```  
  
## Return Value  

 S_OK if the method succeeds; otherwise, E_FAIL or some other error code.  
  
## Requirements  

 **Header:** CorSym.idl, CorSym.h
