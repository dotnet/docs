# IAssemblyEnum::Clone Method

Creates a shallow copy of this [IAssemblyEnum](iassemblyenum-interface.md) object.  
  
## Syntax  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## Parameters  

 `ppEnum`  
 [out] A pointer to the copy.
