# IObjectHandle Interface

Provides a method for unwrapping marshal-by-value objects from indirection.  
  
## Methods  
  
|Method|Description|  
|------------|-----------------|  
|[Unwrap Method](iobjecthandle-unwrap-method.md)|Unwraps a marshal-by-value object from indirection.|
