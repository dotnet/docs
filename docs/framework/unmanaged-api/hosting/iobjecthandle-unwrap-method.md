# IObjectHandle::Unwrap Method

Unwraps a marshal-by-value object from indirection.  
  
## Syntax  
  
```cpp  
HRESULT Unwrap (  
    [out, retval] VARIANT *ppv  
);  
```  
  
## Parameters  

 `ppv`  
 [out] A pointer to the object to be unwrapped.
