# IMetaDataFilter::UnmarkAll Method
Removes the processing marks from all the tokens in the current metadata scope.  
  
## Syntax  
  
```  
HRESULT UnmarkAll ();  
```  
  
#### Parameters  
 `UnmarkAll` takes no parameters.
