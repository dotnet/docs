# IMetaDataImport::ResetEnum Method

Resets the specified enumerator to the specified position.  
  
## Syntax  
  
```cpp  
HRESULT ResetEnum (  
   [in] HCORENUM    hEnum,
   [in] ULONG       ulPos  
);  
```  
  
## Parameters  

 `hEnum`  
 [in] The enumerator to reset.  
  
 `ulPos`  
 [in] The new position at which to place the enumerator.
