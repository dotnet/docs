# ProtocolStopFailure

Id: 147  
  
 Severity: Error  
  
 Category: TransactionBridge  
  
## Description  

 This event indicates that the WS-Atomic Transaction protocol service failed to stop. The event lists the protocol ID, protocol name, exception, process name and process ID.
