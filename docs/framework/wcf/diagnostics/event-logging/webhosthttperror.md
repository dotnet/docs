# WebHostHttpError

Id: 110  
  
 Severity: Error  
  
 Category: ServiceModel  
  
## Description  

 This event indicates that a WebHost HTTP exception occurred. The event lists the Sender Information, HtmlErrorMessage, exception, process name and process ID.
