# Queued Rejected Messages Per Second

Counter Name: Queued Messages Rejected Per Second.  
  
## Description  

 Number of messages that are rejected by the queued transport at this service in a second.  
  
 This counter is of performance counter type [PERF_COUNTER_COUNTER](/previous-versions/windows/it-pro/windows-server-2003/cc740048(v=ws.10)), whose value is calculated using the following formula.  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
