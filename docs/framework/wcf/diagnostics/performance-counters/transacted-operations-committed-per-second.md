# Transacted Operations Committed Per Second
Counter Name: Transacted Operations Committed Per Second.  
  
## Description  
 Number of transactional operations that have been committed in this service in a second.  
  
 This counter is of performance counter type [PERF_COUNTER_COUNTER](https://go.microsoft.com/fwlink/?LinkID=94649), whose value is calculated using the following formula.  
  
 (N 1 - N 0 ) / ( (D 1 -D 0 ) / F)
