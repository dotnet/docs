# System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached
System.ServiceModel.Channels.ConnectionPoolLeaseTimeoutReached  
  
## Description  
 A connection has exceeded the connection lease timeout of this connection pool and been closed.
