# System.ServiceModel.Channels.NamedPipeChannelMessageReceiveFailed

System.ServiceModel.Channels.NamedPipeChannelMessageReceiveFailed  
  
## Description  

 Failed to lookup a channel to receive an incoming message. Either the endpoint or the SOAP action was not found.
