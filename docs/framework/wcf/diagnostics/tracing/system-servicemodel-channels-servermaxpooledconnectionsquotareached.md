# System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached
System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## Description  
 MaxOutboundConnectionsPerEndpoint quota has been reached, so the connection was closed and not reused by the listener.
