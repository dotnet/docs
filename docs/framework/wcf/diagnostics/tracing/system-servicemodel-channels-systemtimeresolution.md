# System.ServiceModel.Channels.SystemTimeResolution

System.ServiceModel.Channels.SystemTimeResolution  
  
## Description  

 The operating system's timer resolution was detected as *n* ticks, which is approximately *m* milliseconds. Where *n* and *m* are numeric values.
