# System.ServiceModel.PortSharing.SharedManagerServiceEndpointNotExist
System.ServiceModel.PortSharing.SharedManagerServiceEndpointNotExist  
  
## Description  
 The shared memory for the endpoint of the specified service does not exist. The service may not be started.
