# System.ServiceModel.WarnHelpPageEnabledNoBaseAddress

System.ServiceModel.WarnHelpPageEnabledNoBaseAddress  
  
## Description  

 The ServiceDebugBehavior Help Page is enabled at a relative address and cannot be created because there is no base address.
