# System.ServiceModel.WarnServiceHealthEnabledNoBaseAddress

System.ServiceModel.WarnServiceHealthEnabledNoBaseAddress  
  
## Description  

 The ServiceHealthBehavior health page is enabled at a relative address and cannot be created because there is no base address.
