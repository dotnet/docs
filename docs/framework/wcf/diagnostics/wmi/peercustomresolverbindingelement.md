## Properties  
 The PeerCustomResolverBindingElement class has the following properties:  
  
### Address  
 Data type: string  
  
 Access type: Read-only  
  
 The address of the peer custom resolver.  
  
### Binding  
 Data type: string  
  
 Access type: Read-only  
  
 The configuration name of the binding.
