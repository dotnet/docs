# 1147 - FlowchartSwitchDefault

| Property | Value |
| - | - |
|ID|1147|

## Description  

 Indicates the default case has been selected in a Flowchart switch.

 Flowchart '%1'/FlowSwitch - Default Case was selected.

|Data Item Name|Data Item Type|Description|
