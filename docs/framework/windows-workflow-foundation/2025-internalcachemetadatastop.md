# 2025 - InternalCacheMetadataStop

|||  
|-|-|  
|ID|2025|  
|Keywords|WFRuntime|  
|Level|Verbose|

## Description  
 Indicates the completion of InternalCacheMetadata on an activity.

 InternalCacheMetadata stopped on activity '%1'.  
  
## Details  
  
|Data Item Name|Data Item Type|Description|  
|--------------------|--------------------|-----------------|
