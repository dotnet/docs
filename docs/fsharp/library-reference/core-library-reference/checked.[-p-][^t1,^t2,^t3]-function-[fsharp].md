# Checked.( + )<^T1,^T2,^T3> Function (F#)

## Syntax

```fsharp

( + ) : ^T1 -> ^T2 -> ^T3 (requires ^T1 with static member (+) and ^T2 with static member (+))

// Usage:
```

*y*
