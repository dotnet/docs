# As Clause (Visual Basic)

Introduces an `As` clause, which identifies a data type in a declaration statement or a constraint list on a generic type parameter.  
  
## Remarks  

 The `As` keyword can be used in these contexts:  
  
 [Aggregate Clause](../queries/aggregate-clause.md)  
  
 [Class Statement](class-statement.md)  
  
 [Const Statement](const-statement.md)  
  
 [Declare Statement](declare-statement.md)  
  
 [Delegate Statement](delegate-statement.md)  
  
 [Dim Statement](dim-statement.md)  
  
 [Enum Statement](enum-statement.md)  
  
 [Event Statement](event-statement.md)  
  
 [For...Next Statements](for-next-statement.md)  
  
 [For Each...Next Statements](for-each-next-statement.md)  
  
 [From Clause](../queries/from-clause.md)  
  
 [Function Statement](function-statement.md)  
  
 [Group Join Clause](../queries/group-join-clause.md)  
  
 [Interface Statement](interface-statement.md)
