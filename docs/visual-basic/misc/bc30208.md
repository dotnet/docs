# &#39;Compare&#39; expected
An `Option` statement specifies `Text` or `Binary`, but not `Compare`.  
  
 **Error ID:** BC30208  
  
## To correct this error  
  
-   Add the `Compare` keyword to the `Option` statement; for example, `Option Compare Binary`.
