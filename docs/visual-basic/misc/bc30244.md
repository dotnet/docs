# &#39;&lt;modifier&gt;&#39; is not valid on a Declare
A `Declare` statement contains an invalid keyword, such as `ReadOnly`.  
  
 **Error ID:** BC30244  
  
### To correct this error  
  
-   Remove the invalid keyword from the `Declare` statement.
