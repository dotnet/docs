# 'Sub New' cannot be declared '\<modifier>'
You have attempted to declare a `Sub New` with a modifier, such as `Overrides`, that is not valid.  
  
 **Error ID:** BC30364  
  
## To correct this error  
  
-   Remove the modifier.
