# Shared &#39;Sub New&#39; cannot be declared &#39;&lt;specifier&gt;&#39;
You have attempted to use a specifier in a `Shared Sub New` declaration, which is not valid.  
  
 **Error ID:** BC30480  
  
## To correct this error  
  
-   Remove the specifier.
