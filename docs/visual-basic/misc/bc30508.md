# '\<name>' cannot expose type \<type1> in \<specifier> \<type2> '\<typename>' through \<name>

You have attempted to expose a private type outside the public class `MyClass`.  
  
 **Error ID:** BC30508  
  
## To correct this error  
  
- Declare the type `Public`.
