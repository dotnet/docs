# Property '\<propertyname>' is 'WriteOnly'

You have tried to retrieve a value from a property that is declared `WriteOnly`.  
  
 **Error ID:** BC30524  
  
## To correct this error  
  
- Remove the `WriteOnly` specifier from the property declaration.
