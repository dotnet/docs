# Property '\<propertyname>' is 'ReadOnly'

You have tried to assign a value to a property that is declared `ReadOnly`.  
  
 **Error ID:** BC30526  
  
## To correct this error  
  
- Remove the `ReadOnly` specifier from the property declaration.
