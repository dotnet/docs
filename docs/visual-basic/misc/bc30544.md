# Method cannot contain both a 'Try' statement and an 'On Error' or 'Resume' statement

You have combined a `Try` statement with `On Error` or `Resume`.  
  
 **Error ID:** BC30544  
  
## To correct this error  
  
1. Remove the `On Error` or `Resume`.
