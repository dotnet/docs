# 'Get' statement must end with a matching 'End Get'
`Get` property procedures must end with an `End Get` statement.  
  
 **Error ID:** BC30631  
  
## To correct this error  
  
-   Make sure that the `Get` property procedure ends with an `End Get` construct.
