# 'ReadOnly' and 'WriteOnly' cannot be combined

The `ReadOnly` and `WriteOnly` modifiers are mutually exclusive.  
  
 **Error ID:** BC30635  
  
## To correct this error  
  
- Specify either `ReadOnly` or `WriteOnly`, but not both.
