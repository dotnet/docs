# 'ByVal' and 'ByRef' cannot be combined

The `ByVal` and `ByRef` keywords are mutually exclusive.  
  
 **Error ID:** BC30641  
  
## To correct this error  
  
- Specify either `ByVal` or `ByRef`, but not both.
