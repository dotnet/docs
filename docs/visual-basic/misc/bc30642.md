# &#39;Optional&#39; and &#39;ParamArray&#39; cannot be combined
The `Optional` and `ParamArray` modifiers are mutually exclusive.  
  
 **Error ID:** BC30642  
  
### To correct this error  
  
-   Specify either `ParamArray` or `Optional`, but not both.
