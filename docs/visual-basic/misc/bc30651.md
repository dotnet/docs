# &lt;Property&gt; parameters cannot be declared &#39;ByRef&#39;
Parameters for properties must be declared using the `ByVal` modifier.  
  
 **Error ID:** BC30651  
  
## To correct this error  
  
-   Define property parameters as `ByVal`.
