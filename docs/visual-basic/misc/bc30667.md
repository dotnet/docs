# ParamArray parameters must be declared &#39;ByVal&#39;
`ParamArray` parameters cannot use the `ByRef` modifier.  
  
 **Error ID:** BC30667  
  
## To correct this error  
  
-   Declare `ParamArray` parameters using the `ByVal` modifier.
