# Expression or statement is not valid in debug windows
`Try...Catch...Finally` statements cannot be used in the debugging context.  
  
 **Error ID:** BC30706  
  
## To correct this error  
  
1.  Remove `Try...Catch...Finally` statements from debugging code.
