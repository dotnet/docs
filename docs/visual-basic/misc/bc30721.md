# Evaluation of expression or statement stopped
The **Stop Evaluation** menu item was selected, causing evaluation to stop.  
  
 **Error ID:** BC30721  
  
## To correct this error  
  
-   Allow execution to continue without selecting **Stop Evaluation**.
