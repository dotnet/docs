# 'Sub Main' is declared more than once in '\<namespace>': \<message>

`Sub Main` can only be declared once inside a namespace.  
  
 **Error ID:** BC30738  
  
## To correct this error  
  
- Make sure there is only a single `Sub Main` procedure in your project.
