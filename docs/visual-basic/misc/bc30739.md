# 'RaiseEvent' statements are not valid in the Immediate window

`RaiseEvent` statements cannot be used in the **Immediate** window.  
  
 **Error ID:** BC30739  
  
## To correct this error  
  
- Test `RaiseEvent` statements from code in the **Code Editor**.
