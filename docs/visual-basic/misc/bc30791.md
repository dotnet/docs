# 'Declare' statements in a structure cannot be declared '&lt;specifier&gt;'
A `Declare` declaration has a specifier that is not valid within a `Structure` declaration.  
  
 **Error ID:** BC30791  
  
## To correct this error  
  
-   Remove the specifier.
