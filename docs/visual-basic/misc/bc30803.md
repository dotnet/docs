# 'Currency' is no longer a supported type; use the 'Decimal' type instead
The `Currency` data type has been replaced by the `Decimal` data type.  
  
 **Error ID:** BC30803  
  
## To correct this error  
  
-   Use the `Decimal` data type instead of `Currency`.
