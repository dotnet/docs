# 'Let' and 'Set' assignment statements are no longer supported

`Let` and `Set` cannot be used in assignments.  
  
 **Error ID:** BC30807  
  
## To correct this error  
  
- Omit `Let` and `Set` from assignment statements; for example:  
  
     `X = 2`
