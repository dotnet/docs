# &#39;Wend&#39; statements are no longer supported; use &#39;End While&#39; statements instead
The `End While` construct replaces the `Wend` keyword.  
  
 **Error ID:** BC30809  
  
## To correct this error  
  
-   Use `End While` to end a `While` block.
