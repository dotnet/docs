# '\<function>' is not declared

'\<function>' is not declared. Function has moved to the Microsoft.VisualBasic namespace.  
  
 **Error ID:** BC30818  
  
## To correct this error  
  
- Use the appropriate function from the `Microsoft.VisualBasic` namespace.
