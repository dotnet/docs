# 'Option Infer' can be followed only by 'On' or 'Off'

`Option Infer` statements must specify either `On` or `Off`. Other values are not valid.  
  
 **Error ID:** BC30979  
  
## To correct this error  
  
- Specify either `Option Infer On` or `Option Infer Off`.
