# Expected closing ']]>' for XML CDATA section
An XML CDATA literal is missing the required closing ']]>' characters.  
  
 **Error ID:** BC31162  
  
## To correct this error  
  
-   Add the required closing ']]>' characters to the end of the XML CDATA literal.
