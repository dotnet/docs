# '\<specifier1>' and '\<specifier2>' cannot be combined

Two incompatible specifiers, such as `Shadows` and `Overrides` are used together.  
  
 **Error ID:** BC31408  
  
## To correct this error  
  
- Remove one or the other specifier.
