# Lambda expression cannot be converted to '&lt;typename&gt;' because type '&lt;typename&gt;' is declared 'MustInherit' and cannot be created
A statement attempts to convert a lambda expression to a type that can only be a base class and cannot be used to create an instance.  
  
 **Error ID:** BC36626
