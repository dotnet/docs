- **`--os <OS>`**

  Specifies the target operating system (OS). This value sets the OS part of the [Runtime Identifier (RID)](../docs/core/rid-catalog.md). If you use this option, don't use `-r|--runtime`. Available since .NET 6 Preview 7.
