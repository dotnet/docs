This sample is part of the [step-by-step tutorial](https://docs.microsoft.com/dotnet/articles/core/tutorials/using-with-xplat-cli)

Key Features
------------

This is the basic Hello World sample. It demonstrates the basics of the environment.

Build and Run
-------------

To build and run the sample, type the following two commands:

`dotnet restore`
`dotnet run`

`dotnet restore` restores the dependencies for this sample.
`dotnet run` builds the sample and runs the output assembly.
