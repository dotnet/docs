C# REST Client Sample
================

This sample is created during the [REST client tutorial](https://docs.microsoft.com/dotnet/articles/csharp/tutorials/console-webapiclient)
for learning C# features. Please see that topic for detailed steps on the code
for this sample.

Key Features
------------

This sample demonstrates making HTTP requests to a web server, using `async`
and `await`, converting JSON objects into C# objects, and terminal output.

Build and Run
-------------

To build and run the sample, type the following two commands:

`dotnet restore`
`dotnet run`

`dotnet restore` restores the dependencies for this sample.
`dotnet run` builds the sample and runs the output assembly.
