C# Iterators Sample
================

This sample is created during the [Iterators topic](https://docs.microsoft.com/dotnet/articles/csharp/iterators)
for learning C# features. Please see that topic for detailed steps on the code
for this sample.

Key Features
------------

This sample demonstrates iterating across a data source, and creating
iterator methods to control how a data source is visited (or iterated).

Build and Run
-------------

To build and run the sample, type the following two commands:

`dotnet restore`
`dotnet run`

`dotnet restore` restores the dependencies for this sample.
`dotnet run` builds the sample and runs the output assembly.
